# vue-scoreboard
An app for keeping player scores. You can add, edit and delete players from the scoreboard!

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
